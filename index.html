WWW - World Wide Web

The world wide web is a collection of HTML documents. At the time of this lesson approximately 30 billion documents make up the web. HTML, or hypertext markup language, is the language of the world wide web. It is the basic building blocks for the documents on the web. When you are browsing the web and you enter a URL into your browser your computer sends an HTTP request to server corresponding to that address. The server repsonds back by sending the file or webpage you requested with the URL.

HTML

Hypertext markup language is the heart of the web. It is made up of text content (what you see on a page), markup (what tells you browser how to arrange the content on the page), references to other documents (images and videos), and links to other pages.

HTML Markup

Markup is the tags and attributes that tell the browser how to display a webpage. It has the following format:

<tag>Contents</tag>

Other examples of tags are the bold tag: <b>
The emphasis tag: <em>
And the line break tag: <br>

Computers are Stupid

A computer can only interpret instructions exactly as you have entered them. It does not infer, and it does not guess.

HTML Attributes

Attributes provide more information about an HTML element. They have the following format:

<tag attr="value">Contents</tag>

Examples include the anchor tag: <a href="URL">Link Text</a>
And the image tag: <img src="URL" alt="TEXT"> (Note that the image tag is a void tag. It does not have contents and does not require a closing tag.)

Whitespace - Inline vs. Block

Inline elements are exactly like they sound. They work inline without breaks. Block elements create an invisible box around its contents and break to a new line.
